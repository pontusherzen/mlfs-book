## Virtual environment
The project is run in a virtual environment, with library versions as specified in requirements.txt.

## Dashboard
The project dashboard can be found at: https://pontusherzen.github.io/mlfs-book/ 

# Create a conda or virtual environment for your project before you install the requirements
    pip install -r requirements.txt

## Project structure
Notebook 1: Data processing
Notebook 2: Daily measurements are collected with lag calculated
Notebook 3: We train two separate models: one for lag and one without lag. Comparing plots are displayed too. 
Notebook 4: Here we make future predictions based on earlier predictions for the coming seven days. With and without lag.
